Historia de Git y GitHub
Git
Creado por: Linus Torvalds (el creador de Linux).
Ao: 2005
Motivo: El equipo de desarrollo del kernel de Linux necesitaba un sistema
de control de versiones
distribuido, rpido y confiable, despus de dejar de usar BitKeeper.
Caractersticas clave:
Distribuido: cada desarrollador tiene una copia completa del
repositorio.
Rpido y eficiente para proyectos grandes.
Seguro: usa hashes SHA-1 para identificar cada cambio.
GitHub
Fundado en: 2008 por Tom Preston-Werner, Chris Wanstrath y PJ Hyett.
Qu es: una plataforma basada en Git que permite alojar repositorios en
la nube, colaborar, revisar
cdigo, gestionar proyectos y ms.
Adquirido por Microsoft en 2018
